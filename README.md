En oversikt over mine prosjekter.
 
Prosjektene under Machine Learning er gruppeprosjekter, der jeg og to andre samarbeidet om å bruke maskinlæring til å analysere datasett. Vi samarbeidet om å skrive koden, og samarbeidet om å skrive rapportene. 

Prosjektet INF102_Algorithms_BigO er en semesteroppgave fra emnet INF102. Fokuset var på algoritmer og kjøretidsanalyse. Vi hadde fått utdelt koden i graph og IProblem. Jeg skrev altså koden i ProblemSolver, i tillegg til å gjennomføre kjøretidsanalyser, som er tilgjengelig i Svar.md.
